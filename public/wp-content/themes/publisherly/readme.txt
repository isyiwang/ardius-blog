== Installation ==

1. Sign into your WordPress dashboard, go to Appearance > Themes, and click Add New button.
2. Click Upload Theme button.
3. Click Choose File button and select the theme zip file you downloaded from Mighty WP.
4. Click Install Now button.
5. After WordPress installs the theme, click Activate so you can use your theme right away.
